Chrome Frame FAQ

Welche Chrome-Versionen werden von diesem Rahmen unterstützt?

Unser Chrome-Rahmen basiert auf dem neuesten Chrome Browser und sorgt so für Konsistenz mit der aktuellen Hauptversion.

Kann ich den Inhalt der Adressleiste anpassen?

Ja! Auf der Hauptseite des Tools können Sie wählen, ob die Adressleiste angezeigt werden soll, und den Adressleisten-Text auf dem generierten Bild bearbeiten.

Gibt es Größenbeschränkungen für generierte Bilder?

Es gibt keine harten Grenzen. Wir unterstützen Bilder von wenigen KB bis zu mehreren Dutzend MB, mit 1x, 2x und 3x Exportgrößen-Optionen.
