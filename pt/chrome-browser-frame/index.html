FAQ do Frame Chrome

Quais versões do Chrome este frame suporta?

Nosso frame Chrome é projetado com base no navegador Chrome mais recente, garantindo consistência com a versão atual mainstream.

Posso personalizar o conteúdo da barra de endereços?

Sim! Na página principal da ferramenta, você pode escolher mostrar ou ocultar a barra de endereços e editar o texto da barra de endereços na imagem gerada.

Existem limites de tamanho para imagens geradas?

Não há limites rígidos. Suportamos imagens de alguns KB a dezenas de MB, com opções de tamanho de exportação 1x, 2x e 3x.
