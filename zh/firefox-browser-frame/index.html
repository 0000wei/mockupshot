Firefox 框架常见问题

这个框架匹配哪个 Firefox 版本？

我们的 Firefox 框架基于最新的 Firefox 稳定版本，采用 2021 年引入的现代 Proton 设计语言。

可以自定义 Firefox 地址栏吗？

可以！在主工具页面，您可以调整地址栏文本，选择是否显示导航元素。

Firefox 框架适合开发者内容吗？

绝对适合。Firefox 在开发者和科技爱好者中具有很强的品牌认知度，使其成为开发工具展示和开源项目演示的理想选择。
