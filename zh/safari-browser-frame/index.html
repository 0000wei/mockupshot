这个框架支持哪个 macOS 版本？

我们的 Safari 框架基于最新的 macOS 版本设计，同时也兼容旧版本 Safari 的界面风格。

该框架主要为 macOS Safari 设计。对于 iOS 设备，我们建议使用移动设备专用框架样式。

Safari 框架包含独特的导航按钮、地址栏设计和 macOS 风格的窗口控制，完美复刻真实的 Safari 浏览器外观。

绝对适合！我们的 Safari 框架遵循 Apple 的设计指南。生成的图片可以安全地用于产品展示、营销材料和任何商业用途。
