Chrome 框架常见问题

这个框架支持哪些 Chrome 版本？

我们的 Chrome 框架基于最新的 Chrome 浏览器设计，确保与当前主流版本保持一致。

可以自定义地址栏内容吗？

可以！在主工具页面，您可以选择显示或隐藏地址栏，并编辑生成图片上的地址栏文本。

生成图片有尺寸限制吗？

没有硬性限制。我们支持从几 KB 到几十 MB 的图片，提供 1x、2x 和 3x 导出尺寸选项。
